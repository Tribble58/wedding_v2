# Добавление изображений

## Детские фотографии жениха и невесты

Чтобы добавить детские фотографии жениха и невесты:

1. Добавьте фотографию жениха в детстве в этот каталог с именем `groom-child.jpg` (или `.png`)
2. Добавьте фотографию невесты в детстве с именем `bride-child.jpg` (или `.png`)
3. Обновите `components/PhotoScreen.tsx`:
   - Установите `hasGroomPhoto = true`
   - Установите `hasBridePhoto = true`

## Изображения для дресс-кода

Чтобы добавить изображения цветовых палитр для дресс-кода:

1. Добавьте изображение с палитрами Black, Olive, Beige с именем `dress-code-colors.jpg` (или `.png`)
2. Добавьте изображение с цветами Graphite, Rain forest, Pistachio, Light gray с именем `dress-code-palette.jpg` (или `.png`)

**Рекомендации по изображениям:**
- Формат: JPG или PNG
- Для детских фото: рекомендуется квадратные изображения 512x512px или больше
- Для изображений дресс-кода: рекомендуется ширина 800-1200px, соотношение сторон сохраняется
- Оптимизируйте изображения для веба перед загрузкой (используйте инструменты типа TinyPNG или ImageOptim)

После добавления изображений они будут автоматически отображаться в приложении.
